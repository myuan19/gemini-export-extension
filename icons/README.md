# 图标文件说明

请在此文件夹中放置以下图标文件：

- `icon16.png` - 16x16 像素
- `icon48.png` - 48x48 像素  
- `icon128.png` - 128x128 像素

## 获取图标的方法

1. **在线生成工具**：
   - https://www.favicon-generator.org/
   - https://realfavicongenerator.net/

2. **设计工具**：
   - 使用 Figma、Photoshop 等设计工具创建
   - 导出为 PNG 格式

3. **临时占位符**：
   - 可以使用任何 16x16、48x48、128x128 的 PNG 图片作为占位符
   - 建议使用简单的文档图标或导出图标

## 图标设计建议

- 使用简洁的设计
- 使用与 Google Gemini 品牌色（蓝色）相近的颜色
- 图标应该在小尺寸下也能清晰识别
- 建议使用文档或导出相关的图标元素
